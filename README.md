# Automation with Whatsapp
Nodejs project bot whatsapp nodejs pokemon. Send number and return pokemon.


# Technologies used.   
body-parser,
dotenv,
puppeteer
nodemon

# Getting Started
1. Clone repo and install npm nodejs git.
2. npm i
3. Run script:

  Chrome
  ```
  node server.js
  ```
  default chromium

  ```
  node app.js
  ```

.env
```

HOST=http://localhost
PORT=3002

```
